5. Soluciones si los iconos no se ven

Si ves cuadrados o X en lugar de iconos:

Solución 1: Verificar la conexión a Internet

Font Awesome se carga desde un CDN. Asegúrate de tener conexión a Internet.

Solución 2: Limpiar caché del navegador

1. Presiona Ctrl + Shift + Delete
2. Marca "Archivos en caché"
3. Haz clic en "Borrar datos"
4. Recarga la página con Ctrl + F5

Solución 3: Verificar que el CDN esté en el HTML

Todos los archivos PHP deben tener esta línea en el <head>:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Solución 4: Descargar Font Awesome localmente

Si el CDN no funciona, descarga Font Awesome:

1. Ve a: https://fontawesome.com/download
2. Descarga la versión gratuita
3. Extrae en: assets/fontawesome/
4. Cambia el link a: <link rel="stylesheet" href="assets/fontawesome/css/all.min.css">
